## 📌 Sobre
Este é um repositório organizado e mantido por **thalissonsenai-boop**. 
Jogo clássico da cobrinha em JavaScript e HTML5 Canvas.

## 🚀 Funcionalidades
- Controle direcional
- Aumento de velocidade progressivo
- Sistema de pontuação

## 🛠️ Tecnologias
Este projeto utiliza:
- **JavaScript (Canvas), HTML, CSS**

## 💻 Como Rodar
1. Clone este repositório:
   ```bash
   git clone https://github.com/thalissonsenai-boop/Snake-game.git
   ```
2. Abra o diretório do projeto:
   ```bash
   cd Snake-game
   ```
3. Abra o `index.html` diretamente no seu navegador.
